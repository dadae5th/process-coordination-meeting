# 엑셀 파일 예시

이 폴더에는 테스트용 엑셀 파일 샘플이 포함되어 있습니다.

## 파일 형식

엑셀 파일은 다음 6개의 칼럼을 포함해야 합니다:

1. **안건** (필수)
2. **해결방안** (선택)
3. **담당자** (필수 - 이메일 형식)
4. **요청부서** (선택 - 이메일 형식)
5. **완료예상일정** (선택 - 날짜 형식)
6. **비고** (선택)

## Excel에서 샘플 파일 만들기

1. Excel 또는 Google Sheets 실행
2. 첫 번째 행에 다음 칼럼 이름 입력:
   ```
   안건 | 해결방안 | 담당자 | 요청부서 | 완료예상일정 | 비고
   ```

3. 두 번째 행부터 데이터 입력 예시:

| 안건 | 해결방안 | 담당자 | 요청부서 | 완료예상일정 | 비고 |
|------|----------|--------|----------|--------------|------|
| 시스템 성능 개선 | API 최적화 및 캐싱 적용 | developer@company.com | planning@company.com | 2025-01-15 | 우선순위 높음 |
| 보안 취약점 점검 | 정기 보안 감사 수행 | security@company.com | it@company.com | 2025-01-20 | 분기별 실시 |
| 사용자 피드백 반영 | UI/UX 개선 작업 | design@company.com | product@company.com | 2025-01-25 | 고객 요청사항 |
| 데이터베이스 마이그레이션 | PostgreSQL 14로 업그레이드 | dba@company.com | it@company.com | 2025-02-01 | 다운타임 최소화 |
| 신규 기능 개발 | 대시보드 차트 추가 | frontend@company.com | product@company.com | 2025-02-10 | 분석 기능 강화 |

4. `.xlsx` 형식으로 저장

## 주의사항

- **담당자 칼럼**: 반드시 유효한 이메일 주소 형식이어야 합니다.
- **요청부서 칼럼**: 이메일 주소를 입력하면 초기 알림이 발송됩니다.
- **완료예상일정**: `YYYY-MM-DD` 형식 권장 (예: 2025-01-15)
- Excel에서 날짜 형식으로 입력해도 자동으로 변환됩니다.

## 테스트 시 주의

처음 테스트할 때는:
- 담당자와 요청부서에 **본인의 이메일 주소**를 입력하세요.
- Resend 무료 플랜에서는 인증되지 않은 도메인의 경우, 자신의 이메일로만 발송 가능합니다.

## 업로드 방법

1. 대시보드 접속
2. "파일 선택" 버튼 클릭 또는 드래그 앤 드롭
3. 엑셀 파일 선택
4. 자동으로 업로드 및 파싱
5. "이메일 발송" 버튼 클릭하여 초기 알림 전송
